# Bright-TV-Case-Study
Bright TV – Data Analysis Project
Overview

This project analyzes Bright TV viewership and user behaviour data to understand consumption patterns across different demographics, provinces, and content channels. The goal is to generate insights that help improve user engagement and optimize content strategy.

Objectives
Analyze user demographics (age, gender, race, province)
Understand TV channel consumption trends
Identify high and low performing content areas
Compare user engagement across different segments
Support data-driven decisions for content improvement
Dataset Description

The dataset includes the following fields:

UserID – Unique identifier for each viewer
Gender – Male / Female / Other
Age – Age of the user
Race – Demographic category
Province – Location of the user
Channel – TV channel watched
RecordDate – Date of viewing activity
Data Preparation
Handled missing values using IFNULL() / default values
Standardized categorical fields (Gender, Province, Channel)
Rounded numerical values where needed
Cleaned inconsistent text entries
Key Analyses Performed
User count by gender and age group
Viewership distribution by province
Most and least watched channels
Daily and weekly consumption trends.
